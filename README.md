## 技術スタック
### フロントエンド：React/TypeScript/Tailwind CSS
### バックエンド：Hasura

</br>

## ローカル環境

パッケージのインストール

```
yarn
```

画面起動

```bash
yarn start
```

</br>

## 環境変数の設定

環境変数準備

```bash
cp .env.example .env
```

```bash
REACT_APP_FIREBASE_API_KEY=
```
